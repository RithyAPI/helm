Thank you for installing {{ include "oracle-db.fullname" . }}!

To connect from inside the cluster:
  SERVICE  : {{ include "oracle-db.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local
  PORT     : {{ .Values.service.port | default 1521 }}
  SERVICE NAME: {{ .Values.env.ORACLE_DATABASE | default "XEPDB1" }}

Example:
  sqlplus sys/<password>@//{{ include "oracle-db.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local:{{ .Values.service.port | default 1521 }}/{{ .Values.env.ORACLE_DATABASE | default "XEPDB1" }} as sysdba

If service.type = NodePort, use any node IP + nodePort shown by:
  kubectl -n {{ .Release.Namespace }} get svc {{ include "oracle-db.fullname" . }} -o wide
